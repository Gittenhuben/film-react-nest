# Бэкенд проекта "Film!"
**Выполненная работа:**
* Разработка бэкенда (NestJS, MongoDB, PostgreSQL, Nginx) с возможностью выбора СУБД в .env
* Докеризация (Docker)
* Деплой (Yandex Cloud)
* Реализация Continuous Delivery (GitHub Actions)

**Запуск:**
* `docker-compose -f docker-compose-local.yml up`
